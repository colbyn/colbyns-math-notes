== §3.5 | Inverse Trigonometric Functions

.Definition of stem:[\arcsin(x)]
****
[latexmath]
++++
y = \arcsin(x)
++++
if and only if
[latexmath]
++++
x = \sin(y)
++++
where
[latexmath]
++++
-1 \leq x \leq 1 \\
-\frac{1}{4}\tau \leq y \leq \frac{1}{4}\tau
++++

.Definition of stem:[\arccos(x)]
****
[latexmath]
++++
y = \arccos(x)
++++
if and only if
[latexmath]
++++
x = \cos(y)
++++
where
[latexmath]
++++
-1 \leq x \leq 1 \\
